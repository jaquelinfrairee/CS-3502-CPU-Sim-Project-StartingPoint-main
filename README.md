# CPU Scheduling Simulator

## Description
This project implements CPU scheduling algorithms including:
- FCFS
- SJF
- Round Robin
- Priority
- SRTF (added)
- HRRN (added)

## Platform
Windows (Visual Studio)

## How to Run
1. Open the solution in Visual Studio
2. Build the project
3. Run the application

## Features
- Average Waiting Time
- Average Turnaround Time
- CPU Utilization
- Throughput
- Final Submission
